== Description ==
This plugin is a simple, reliable and quick solution for using "question-answer"-blocks with the gutenberg editor. It can be used perfectly for "Frequently Asked Questions (FAQ)".

== Installation ==

= Automatic installation =

Just install from your WordPress "Plugins > Add New" screen. In the search field type “'Vie FAQ Collapsible Dropdown,”. As soon as the plugin appears it is ready for installation. Click “'Install Now,” and activate it.

= Manual installation =

Manual installation method requires downloading the 'Vie FAQ Collapsible Dropdown' plugin and uploading it to your web server via your favorite FTP application. Find more information in the [manual installation instructions](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
- Upload the zip file and unzip it in the `/wp-content/plugins/` directory
- Activate the plugin through the 'Plugins' menu in WordPress `/wp-admin/plugins.php`

As soon as the plugin in is installed, you can add FAQ blocks in your Gutenberg editor.
